
Installation von JonDoFox
-------------------------

1. Oeffnen Sie ein Terminal-Fenster und wechseln Sie in das Verzeichnins, in dem die
   Datei "install_linux.sh" liegt. (Das Verzeichnis, das Sie soeben entpackt haben):

      user@linux:~# cd <unpacked folder>

2. Fuehren Sie nun das Installationsskript aus, indem Sie eingeben:

      user@linux:<unpacked folder># ./install_linux.sh

Wenn Sie bereits JonDoFox installiert haben, werden Sie gefragt, ob Sie Ihre vorhandene 
Installation ueberschreiben wollen. Ihre Bookmarks bleiben dabei erhalten.

WICHTIG: Starten Sie das Linux-Installationsskript NICHT mit einem DOPPELKLICK 
auf die Datei! Fuehren Sie dieses Skript NUR in einem TERMINAL aus!


Deinstallation von JonDoFox
---------------------------

Starten Sie den Profilmanager von Firefox (im Terminal mit der Option -P )

      user@linux:~# firefox -P

Im Profilmanager können Sie das Profil "JonDoFox" löschen.
