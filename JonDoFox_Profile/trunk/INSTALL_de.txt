
Installation von JonDoFox
-------------------------

Entpacken Sie das Archiv entweder mit einem Rechtsklick im Dateimanager oder auf
der Kommandozeile im Terminal mit folgendem Befehl:

   user@linux:~# tar -xjf jondofox_linux_bsd.tar.bz2

Oeffnen Sie ein Terminal-Fenster und wechseln Sie in das Verzeichnins, in dem die
Datei "install_jondofox.sh" liegt. (Das Verzeichnis, das Sie soeben entpackt haben):

   user@linux:~# cd jondofox_linux_bsd

Fuehren Sie nun das Installationsskript aus, indem Sie eingeben:

   user@linux:<unpacked folder># ./install_jondofox.sh

Wenn Sie bereits JonDoFox installiert haben, werden Sie gefragt, ob Sie Ihre vorhandene 
Installation ueberschreiben wollen. Ihre Lesezeichen bleiben dabei erhalten.

WICHTIG: Wenn Sie das Linux-Installationsskript ausfuehren wollen, machen Sie KEINEN DOPPELKLICK 
auf die Datei "install_linux.sh"! Fuehren Sie dieses Skript NUR in einem TERMINAL aus!
